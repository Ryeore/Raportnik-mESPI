# Ekrany UI (MVP)

1. **Auth** — login/rejestracja, Google/Apple.
2. **Feed** — lista raportów grupowana po dniach, infinite scroll, pull-to-refresh, badge ESPI/EBI.
3. **Filters** — bottom-sheet: obserwowane, typ, zakres dat, słowa kluczowe.
4. **Companies** — wyszukiwarka, gwiazdka obserwowania.
5. **Report detail** — tytuł, emitent, numer, data, treść, mark-read, share.
6. **Profile** — obserwowane, historia, ustawienia push (immediate/daily, słowa kluczowe).

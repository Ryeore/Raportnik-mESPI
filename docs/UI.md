# UI screens (MVP)

1. **Auth** — login/registration, Google/Apple.
2. **Feed** — report list grouped by day, infinite scroll, pull-to-refresh, ESPI/EBI badge.
3. **Filters** — bottom-sheet: watched, type, date range, keywords.
4. **Companies** — search, follow star.
5. **Report detail** — title, issuer, number, date, body, mark-read, share.
6. **Profile** — watchlist, history, push settings (immediate/daily, keywords).
